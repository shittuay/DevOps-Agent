Welcome! Let's get started

To use the DevOps Agent, you need an Anthropic API key. This key allows the agent to connect to Claude AI.

How to get your API key:

  1. Visit console.anthropic.com
  2. Sign up or log in to your account
  3. Navigate to "API Keys" section
  4. Create a new API key
  5. Copy and paste it below
Format: sk-ant-api03-...

Your API key is stored securely in config/.env and never shared.
You can change it anytime by editing the file directly.
